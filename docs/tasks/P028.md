# P028: 优化区域过滤正则（同步三文件）

## 背景

P027 仅更新了 ProMax.yaml（hutao-前缀）的过滤正则，Standard.yaml 和 Pro.yaml 仍为旧版简单正则。新版正则进一步优化了关键词、修复了错误、消除了歧义。

## 变更内容

### 命名调整

Standard.yaml / Pro.yaml 使用 `x-` 前缀锚点名，ProMax.yaml 使用 `hutao-`。三文件保持各自前缀不变。

| 文件 | 前缀示例 |
|---|---|
| Standard.yaml | `x-filter-hk: &filter-hk` |
| Pro.yaml | `x-filter-hk: &filter-hk` |
| ProMax.yaml | `hutao-filter-hk: &filter-hk` |
