# P015: Embydirect 策略组嵌套 China 选项

## 任务目标

将 `China` 策略组加入 `Embydirect` 的 proxies 列表，使其可选择国内直连路由。

## 修改范围

| 文件 | 操作 |
|---|---|
| `mihomo/config/ProMax.yaml` | Embydirect 策略组：展开 `*direct` 锚点引用为内联列表，追加 `China` |

## 修改前

```yaml
  - { name: Embydirect, type: select, proxies: *direct, icon: https://cdn.jsdelivr.net/gh/Koolson/Qure@master/IconSet/Color/Emby.png }
```

## 修改后

```yaml
  - { name: Embydirect, type: select, proxies: [DIRECT, 故障转移, 节点选择, 香港节点, 台湾节点, 日本节点, 狮城节点, 韩国节点, 美国节点, China], icon: https://cdn.jsdelivr.net/gh/Koolson/Qure@master/IconSet/Color/Emby.png }
```

## 说明

- 不能直接在 `&direct` 锚点追加 `China`，因为 `China` 策略组本身 `proxies: *direct` 会形成循环引用
- 改为展开 `*direct` 的内联列表，在末尾追加 `China`
- 其他引用 `*direct` 的组不受影响

## 验证要求

```bash
python -c "import yaml; yaml.safe_load(open('mihomo/config/ProMax.yaml','r',encoding='utf-8')); print('YAML OK')"

# 验证 Embydirect 包含 China 选项
grep 'Embydirect' mihomo/config/ProMax.yaml | grep 'China'
```
