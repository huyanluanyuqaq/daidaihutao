# 任务 P025：统一锚点键名前缀 x → hutao

## 描述

在 `mihomo/config/ProMax.yaml` 中将所有 `x-` 前缀的锚点键名统一改为 `hutao-` 前缀，同时保持锚点名 `&foo` 和引用 `*foo` 不变。

## 背景

当前配置文件中锚点键名统一使用 `x-` 前缀（如 `x-base-provider`、`x-url-test` 等），项目希望统一使用 `hutao-` 前缀以体现品牌标识。

## 修改内容

### 文件列表

| 文件 | 变更类型 |
|---|---|
| `mihomo/config/ProMax.yaml` | 键名前缀替换 |

### 注意事项

1. **仅替换键名**（如 `x-base-provider:`），**不替换锚点名**（`&base-provider`）和**引用**（`*base-provider`）
2. 保持 YAML 缩进不变
3. 保持 YAML 语法正确
