# 任务 P024：补齐直连规则集的 fake-ip-filter

## 描述

在 `mihomo/config/ProMax.yaml` 的 `dns.fake-ip-filter` 段补齐直连/国内相关规则集的 `real-ip` 条目：
- 新增 `RULE-SET,Steam-cn,real-ip`
- 新增 `RULE-SET,Cusdirect,real-ip`

## 背景

`fake-ip-filter-mode: rule` 下，`fake-ip-filter` 决定哪些域名不返回 fake-ip、而是解析真实 IP。

P023 新增了 `RULE-SET,Steam-cn,China`，但 `Steam-cn` 未加入 `fake-ip-filter`。同时现有 `RULE-SET,Cusdirect,Cusdirect` 是明确直连语义，也未加入 `fake-ip-filter`。为保持直连规则集 DNS 行为一致，需要补齐这两个规则集。

## 修改内容

### 文件列表

| 文件 | 变更类型 |
|---|---|
| `mihomo/config/ProMax.yaml` | 新增 fake-ip-filter 条目 |

### 具体变更

| # | 位置 | 操作 | 说明 |
|---|---|---|---|
| 1 | `dns.fake-ip-filter` 段 | 新增 | 添加 `RULE-SET,Cusdirect,real-ip`，置于 `Embydirect` 下方或同类直连规则附近 |
| 2 | `dns.fake-ip-filter` 段 | 新增 | 添加 `RULE-SET,Steam-cn,real-ip`，置于 `Ai-cn` 下方或同类国内规则附近 |

### 修改后效果

```
修改前：
    - RULE-SET,Selfdirect,real-ip
    - RULE-SET,Embydirect,real-ip
    - RULE-SET,China,real-ip
    - RULE-SET,Ai-cn,real-ip

修改后：
    - RULE-SET,Selfdirect,real-ip
    - RULE-SET,Embydirect,real-ip
    - RULE-SET,Cusdirect,real-ip
    - RULE-SET,China,real-ip
    - RULE-SET,Ai-cn,real-ip
    - RULE-SET,Steam-cn,real-ip
```

## 注意事项

1. 仅修改 `ProMax.yaml`
2. 添加的是 **规则集名称**：`Cusdirect`、`Steam-cn`
3. 不添加策略组名称；`fake-ip-filter` 条目格式保持 `RULE-SET,<规则集>,real-ip`
4. 保持 YAML 缩进一致
