# 任务 P020：移除 rule-providers 中的 proxy 字段

## 描述

当前三个配置文件（Standard.yaml / Pro.yaml / ProMax.yaml）的 `rule-providers` 段中，每个规则集引用都包含 `proxy: China` 字段，指定使用 China（直连）下载规则集文件。去除该字段后，下载行为由 mihomo 的默认策略决定（通常走自动选择或当前活动代理），简化配置。

`proxy-providers` 段中的 `proxy` 字段 **保留不变**。

## 背景

mihomo 的 `rule-providers` 支持 `proxy` 字段指定下载规则集文件所用的代理。在现有配置中，所有规则集都被设为 `proxy: China`（直连下载）。实际使用中，规则集文件通常托管在 GitHub 等海外服务上，若网络环境限制直连到 GitHub，直接下载可能失败。

去除 `proxy: China` 后，下载行为回退到 mihomo 默认行为（由 `proxy-groups` 中的策略组自动选择），在某些网络环境下更为鲁棒。

### 修改模式

将 rule-providers 段中每条规则集引用末尾的 `, proxy: China` 移除。

**修改前：**

```yaml
Private:    { <<: *rule-set-domain, url: https://..., proxy: China }
```

**修改后：**

```yaml
Private:    { <<: *rule-set-domain, url: https://... }
```

### 排除范围

- `proxy-providers` 段中的 `proxy: China` **不得修改**
- 所有其他配置内容保持不变
