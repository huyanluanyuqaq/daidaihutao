# P003: OpenClash 覆写适配 + README 审查标准

## 任务目标

1. 将 `mihomo/openclash/` 目录下的 4 个覆写文件精简为 3 个，名称与 `mihomo/config/` 的配置文件对应
2. 更新覆写文件内的 DOWNLOAD_FILE/CONFIG_FILE/ruby_map_edit 引用，适配当前仓库的实际配置名
3. 取消 `mihomo/config/` 中 proxy-providers 的注释，使 OpenClash 的 ruby_map_edit 能在运行时找到 provider 条目
4. 为 ProMax.yaml 新增 Landing 第三订阅槽位，适配 OneSmartPro 覆写的 3 个 EN_KEY
5. 更新所有关联文档（README.md、mihomo/config/README.md）
6. 在 PROJECT-SPEC.md 的推送前审查中添加 README 一致性审查项

## 修改范围

### 修改的文件

**mihomo/openclash/**（由 DeveCo 执行，涉及文件重命名和内容更新）：
| 旧文件 | 操作 | 新文件 |
|--------|------|--------|
| `MihomoPro_overwrite.conf` | 重命名 + 更新引用 | `Pro_overwrite.conf` |
| `OneSmartPro_overwrite.conf` | 重命名 + 更新引用 | `ProMax_overwrite.conf` |
| `OneTouch_overwrite.conf` | 重命名 + 更新引用 | `Standard_overwrite.conf` |
| `OneSmart_overwrite.conf` | 删除 | — |

- DOWNLOAD_FILE URL 中的文件名改为对应实际配置（如 `Pro.yaml` 而非 `MihomoPro.yaml`）
- CONFIG_FILE 路径同步更新
- ruby_map_edit 中的 provider 名称从"优质服务商/备用服务商/落地服务商"改为 `Primary/Backup/Landing`

**mihomo/config/：**
- `Pro.yaml` — 取消注释 proxy-providers（Primary, Backup）
- `ProMax.yaml` — 取消注释 + 新增 Landing 条目（3 槽）
- `Standard.yaml` — 取消注释 proxy-providers（Primary, Backup）

**文档文件：**
- `PROJECT-SPEC.md` — 推送前审查新增第 7 条 README 审查
- `README.md` — 更新目录树（移除不存在的 `mihomo/image/` 和 `notes/`）、更新 OpenClash 对照表
- `mihomo/config/README.md` — 快速上手示例增加 Landing 说明

```
mihomo/config/ 除 proxy-providers 外的其他配置

```

## 禁止修改项

- 不得修改 `docs/tasks/`、`docs/reviews/`、`docs/archive/` 中的任务/Review/归档文件
- 不得修改 PROJECT-SPEC.md 除"推送前规范一致性审查"章节以外的内容
- 不得修改 `mihomo/config/` 中的策略组、规则、DNS、TUN 等配置
- 不得修改覆写文件中的 `DISABLE_UDP_QUIC`、更新定时、SUB_INFO_URL 等基础设置

## 验收标准

1. `mihomo/openclash/` 目录下只剩 3 个文件：`Pro_overwrite.conf`、`ProMax_overwrite.conf`、`Standard_overwrite.conf`
2. 每个覆写文件的 DOWNLOAD_FILE URL 中的文件名与 CONFIG_FILE 名称一致
3. 每个覆写文件的 CONFIG_FILE 路径指向实际存在的配置文件
4. ruby_map_edit 中的 provider 名称与配置文件中 proxy-providers 下的条目名称一致
5. ProMax.yaml 的 proxy-providers 包含 Primary、Backup、Landing 三个条目
6. Pro.yaml 和 Standard.yaml 的 proxy-providers 包含 Primary、Backup 两个条目
7. 三个配置文件的 proxy-providers 均未注释
8. README.md 目录树不再出现 `mihomo/image/` 和 `notes/`
9. README.md 的 OpenClash 表格显示 3 个文件及对应目标配置
10. PROJECT-SPEC.md 推送前审查包含 README 审查项（第 7 条）
