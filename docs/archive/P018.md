# 任务 P018：修正 ProMax.yaml rules 区域注释与段落编号

## 描述

P017 新增规则后，对 `mihomo/config/ProMax.yaml` 的 `rules` 区域进行了手动规则排序重排，导致段落注释编号错乱（9 号出现在 8 号之前）及部分注释表述不准确。本任务修正这些注释问题，保持与实际规则顺序一致。

## 分析

### 当前 rules 区域章节顺序（手动重排后）

```

# 8. 通用国外流量           ← 出现在 9 之后，编号错乱
# 10. 兜底规则
```
